## Overview

SSC English Testverse is an Android application developed for SSC aspirants to practice the English section through structured mock tests and topic-wise preparation.

The primary goal was to build a distraction-free testing environment with structured analytics and scalable content management.

---

## The Requirement

The client wanted:

- A dedicated English-focused test platform  
- Real exam-like mock simulations  
- Topic-wise test structure  
- Detailed performance analysis  
- No ads or promotional clutter  

The app needed to feel calm, focused, and serious â€” aligned with exam preparation.

---

## Solution Approach

The application was built with a modular architecture separating:

- Test engine logic  
- Content management  
- User progress tracking  
- Notification workflows  

Content is managed dynamically using a CMS, allowing the client to add and manage question sets without app updates.

---

## Core Features

- 5000+ test sets  
- 15,000+ English questions  
- Topic-wise practice modules  
- Full mock exam simulation  
- Vocabulary study section  
- Strength & weakness tracking  
- Clean and distraction-free interface  

---

## Architecture

### Frontend
- Built using Expo (React Native)
- Optimized navigation flow for test attempts
- Modular component structure for scalability

### Backend
- Supabase (PostgreSQL)
- Authentication
- Result storage
- User progress tracking

### Content Management
- Sanity CMS
- Structured schema for:
  - Exams
  - Topics
  - Difficulty levels
  - Question grouping

### Notifications
- OneSignal integration for engagement-based reminders

---

## Technical Challenges

### Large Question Dataset Handling
Implemented controlled data fetching and structured pagination to maintain performance while handling thousands of questions.

### Scoring Logic
Designed a real-time scoring engine calculating:
- Accuracy percentage
- Sectional performance
- Historical test comparison

### CMS Structuring
Created flexible schemas to support multiple SSC exam categories and scalable content expansion.
